用SSM开发的一个学校教材订购系统
=============

作者 :jianghuan

qq :2959543063

系统介绍
-----------

本项目使用SSM开发的学校教材订购系统，基本的功能有登录，注册，教材的增删改查，图书订购和图书采购等功能

开发工具
-----------

IntelliJ IDEA + Navicat + Git + Chrome

开发技术
-----------

前端：jQuey + ajax

后端：SSM + MySQL
